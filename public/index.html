Timestamp Microservice API

Usage:

1. Pass in a date as a string parameter in the URL. It will check to see whether that string contains either a unix timestamp or a natural language date.
2. If the string is a valid date or Unix timestamp, it will return both the Unix timestamp and the natural language form of that date.
3. If the string is not a valid date or Unix timestamp, it will return null for those properties.

Example Usage:

<url>/1467844008
http://timestamp-ytv.herokuapp.com/June 6, 2016
http://timestamp-ytv.herokuapp.com/Jun 6 16
http://timestamp-ytv.herokuapp.com/6 6 16

Example Output:

{"unix":1467844008,"natural":"June 6, 2016"}
